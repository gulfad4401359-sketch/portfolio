Project 08

Real-Time Collaboration Platform

WebSocketsReact CRDTsCanvas API

Aug 2021 — Jan 2022

Project Overview

Engineered a real-time collaborative whiteboard platform supporting simultaneous editing by up to 50 users per session. The platform uses Conflict-free Replicated Data Types (CRDTs) to guarantee eventual consistency without central locking and serves 25,000 remote teams globally.

Challenge

Existing approaches using Operational Transformation introduced server-side complexity and became a bottleneck under high concurrency. Canvas rendering performance degraded sharply beyond 20 simultaneous cursors. The system also needed to gracefully handle network partitions and reconnections without data loss.

Solution

Implemented a CRDT-based sync engine using Yjs, eliminating the need for server-side conflict resolution. Built a custom Canvas renderer with spatial partitioning to maintain 60fps at 50 concurrent cursors. Designed a connection recovery protocol that replays missed operations from a server-side event log.
