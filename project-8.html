Project 08

Real-Time Collaboration Platform

WebSocketsReact CRDTsCanvas API

Aug 2021 — Jan 2022

Project Overview

Real-time collaborative whiteboard platform supporting up to 50 simultaneous users per session using CRDTs for guaranteed eventual consistency. Serves 25,000 remote teams globally.

Challenge

Canvas rendering performance degraded sharply beyond 20 simultaneous cursors. The system needed to gracefully handle network partitions and reconnections without data loss.

Solution

Implemented a CRDT-based sync engine using Yjs. Built a custom Canvas renderer with spatial partitioning to maintain 60fps at 50 concurrent cursors. Connection recovery protocol replays missed operations from a server-side event log.
