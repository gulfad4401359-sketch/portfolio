Project 02

Design System & Component Library

Design Systems Figma Storybook CSS

Apr 2023 — Aug 2023

Project Overview

Built a complete design system and component library that unified three separate products — a SaaS dashboard, a marketing site, and a mobile app — under a single source of truth. The system covers 120+ components across 8 semantic categories.

Challenge

Each product had evolved independently for three years, resulting in three separate Figma libraries, two competing CSS frameworks, and no shared token layer. Designers and developers spent an estimated 40% of their time resolving inconsistencies rather than shipping features.

Solution

Established a token-first architecture — semantic design tokens in Figma synced directly to CSS custom properties via Style Dictionary. Built a Storybook documentation site with live component previews, accessibility reports, and copy-paste usage snippets. Ran weekly design-dev syncs to maintain adoption momentum.

Tech Stack
Figma Storybook Style Dictionary CSS
Year 2023
Role Design System Lead

Key Outcomes

  • Design-to-dev handoff time reduced by 60% — from an average of 3 days to 1.2 days per feature
  • 120+ components documented in Storybook with full accessibility annotations and usage guidelines
  • Three products visually unified within two quarters of system adoption
