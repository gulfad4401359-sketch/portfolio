Project 02

Design System & Component Library

Design SystemsFigma StorybookCSS

Apr 2023 — Aug 2023

Project Overview

Built a complete design system and component library that unified three separate products under a single source of truth. The system covers 120+ components across 8 semantic categories.

Challenge

Each product had evolved independently for three years, resulting in three separate Figma libraries and two competing CSS frameworks. Designers and developers spent an estimated 40% of their time resolving inconsistencies rather than shipping features.

Solution

Established a token-first architecture — semantic design tokens in Figma synced directly to CSS custom properties via Style Dictionary. Built a Storybook documentation site with live component previews and accessibility reports.

  • Three products visually unified within two quarters of system adoption
