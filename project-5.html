Project 05

SaaS Onboarding Flow

Product DesignVue.js A/B TestingAnalytics

Jun 2022 — Aug 2022

Project Overview

Redesigned the trial onboarding experience for a B2B project management SaaS with 14,000 trial signups per month. The goal was to reduce time-to-first-value and increase the 30-day trial-to-paid conversion rate.

Challenge

The existing onboarding required users to configure 11 settings before they could create their first project. 74% of users abandoned during this setup phase, never experiencing the product's core value. The average time to first meaningful action was 23 minutes.

Solution

Implemented a "jobs-to-be-done" framework to identify the one action that best predicted long-term retention. Rebuilt onboarding around that single activation moment — creating a first project — achievable in under 90 seconds. All configuration moved to post-activation progressive prompts.

Tech Stack
Vue.jsMixpanel LaunchDarklyFigma
Year2022
RoleProduct Designer

Key Outcomes

  • Trial-to-paid conversion rate increased by 52% — from 8.3% to 12.6%
  • Time to first meaningful action reduced from 23 minutes to under 90 seconds
  • Onboarding abandonment dropped from 74% to 18% in the first 4 weeks post-launch
