Project 01

E-Commerce Platform Redesign

UX Design React Node.js TypeScript

Jan 2024 — Mar 2024

Project Overview

A full-scale redesign of a mid-market e-commerce platform serving 200,000 monthly active users. The goal was to reduce cart abandonment by 30% and improve mobile conversion rates through a complete overhaul of the user experience and technical architecture.

Working directly with the product and engineering teams, this project spanned 12 weeks from discovery to launch — encompassing research, design, development, and quality assurance across all touchpoints.

Challenge

The existing platform had accumulated five years of technical debt. Navigation was inconsistent across devices, the checkout flow required seven steps with multiple page reloads, and the design system had fragmented into three divergent component libraries maintained by different teams. Mobile users saw a conversion rate of 1.8% against an industry benchmark of 3.5%.

Solution

Designed and built a unified component system from scratch, collapsed the checkout flow from seven steps to three, and introduced a persistent cart sidebar that eliminated full page transitions. Implemented progressive disclosure to surface relevant product information at the right moment without overwhelming the user.

The new architecture used React with TypeScript, a dedicated design token layer, and a Node.js BFF (Backend for Frontend) that reduced average API response times from 480ms to 90ms.

Key Outcomes

  • Cart abandonment reduced by 34% in the first quarter post-launch, exceeding the original 30% target
  • Mobile conversion rate increased from 1.8% to 3.2% — a 78% improvement against the 3.5% industry benchmark
  • Page load time reduced from 4.2s to 1.1s through code splitting, lazy image loading, and BFF optimisation
