Project 01

E-Commerce Platform Redesign

UX DesignReact Node.jsTypeScript

Jan 2024 — Mar 2024

Project Overview

A full-scale redesign of a mid-market e-commerce platform serving 200,000 monthly active users. The goal was to reduce cart abandonment by 30% and improve mobile conversion rates through a complete overhaul of the user experience and technical architecture.

Challenge

The existing platform had accumulated five years of technical debt. Navigation was inconsistent across devices, the checkout flow required seven steps with multiple page reloads, and the design system had fragmented into three divergent component libraries. Mobile users saw a conversion rate of 1.8% against an industry benchmark of 3.5%.

Solution

Designed and built a unified component system from scratch, collapsed the checkout flow from seven steps to three, and introduced a persistent cart sidebar. The new architecture used React with TypeScript and a Node.js BFF that reduced average API response times from 480ms to 90ms.

Key Outcomes
