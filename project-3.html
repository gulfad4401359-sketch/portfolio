Project 03

AI-Powered Analytics Dashboard

ReactPython D3.jsMachine Learning

Sep 2023 — Dec 2023

Project Overview

Real-time analytics dashboard for a Series A fintech startup, integrating ML models to surface actionable insights from 50M+ daily transaction records.

Challenge

Reports were generated manually in Excel, taking up to 4 hours per analyst per day. Decisions were being made on week-old data.

Solution

Built a streaming data pipeline using Python FastAPI and WebSockets delivering real-time chart updates. Custom D3.js visualisations with natural language insight summaries generated by a fine-tuned LLM.
