Project 03

AI-Powered Analytics Dashboard

React Python D3.js Machine Learning

Sep 2023 — Dec 2023

Project Overview

Designed and engineered a real-time analytics dashboard for a Series A fintech startup, integrating machine learning models to surface actionable insights from 50M+ daily transaction records. The dashboard replaced three separate reporting tools with a single, unified interface.

Challenge

The client's data science team had built accurate predictive models but had no interface for non-technical stakeholders to interact with them. Reports were generated manually in Excel, taking up to 4 hours per analyst per day. Decisions were being made on week-old data.

Solution

Built a streaming data pipeline using Python FastAPI and WebSockets to deliver real-time chart updates. Implemented custom D3.js visualisations with natural language insight summaries generated by a fine-tuned LLM. Analyst manual reporting time dropped to zero.

Tech Stack
React Python FastAPI D3.js PostgreSQL
Year2023
RoleFull-Stack Engineer

Key Outcomes

  • Analyst manual reporting time reduced from 4 hours/day to zero — fully automated
  • Decision latency reduced from 7 days (weekly exports) to under 5 minutes (real-time streaming)
  • Dashboard adopted by all 12 non-technical stakeholders within 2 weeks of launch
