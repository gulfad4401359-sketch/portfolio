Project 01

E-Commerce Platform Redesign

UX Design React Node.js

Jan 2024 — Mar 2024

Project Overview

A full-scale redesign of a mid-market e-commerce platform serving 200k monthly active users. The goal was to reduce cart abandonment by 30% and improve mobile conversion rates.

Challenge

The existing platform had accumulated 5 years of technical debt. Navigation was inconsistent across devices, checkout flow required 7 steps, and the design system was fragmented across 3 divergent component libraries.

Solution

Designed and built a unified component system, collapsed checkout to 3 steps, and introduced a persistent cart sidebar. Implemented progressive disclosure to surface relevant product information without overwhelming the user.

Tech Stack
React TypeScript Node.js
Year 2024
Role Lead Designer & Developer
↗ Live Link ⌥ GitHub

Key Outcomes

  • Cart abandonment reduced by 34% in the first quarter post-launch
  • Mobile conversion rate increased from 1.8% to 3.2% — a 78% improvement
  • Page load time reduced from 4.2s to 1.1s through code splitting and image optimisation
