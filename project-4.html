Project 04

Mobile Banking App Redesign

iOSAndroid React NativeUX Research

Feb 2023 — May 2023

Project Overview

Led the end-to-end redesign of a mobile banking application for a regional bank with 850,000 customers. The project encompassed UX research, interaction design, accessibility remediation, and React Native implementation across iOS and Android.

Challenge

The existing app held a 2.9-star rating with 4,200 reviews primarily citing confusing navigation and security friction. The onboarding flow had a 68% drop-off rate. Biometric authentication was broken on 23% of Android device models.

Solution

Conducted 24 user interviews across three customer segments. Rebuilt navigation around a bottom tab bar with persistent account summary. Replaced the 9-step onboarding with a 4-step progressive flow. Fixed 47 biometric authentication edge cases through systematic device matrix testing.

Tech Stack
React NativeTypeScript FigmaJest
Year2023
RoleLead UX Engineer

Key Outcomes

  • App Store rating improved from 2.9 to 4.8 stars within 8 weeks of launch
  • Onboarding completion rate increased from 32% to 89% — a 178% improvement
  • Daily active users grew by 40% over the following quarter
