Project 04

Mobile Banking App Redesign

iOSAndroid React NativeUX Research

Feb 2023 — May 2023

Project Overview

End-to-end redesign of a mobile banking application for a regional bank with 850,000 customers, encompassing UX research, interaction design, accessibility remediation, and React Native implementation.

Challenge

The existing app held a 2.9-star rating. The onboarding flow had a 68% drop-off rate. Biometric authentication was broken on 23% of Android device models.

Solution

Conducted 24 user interviews. Rebuilt navigation around a bottom tab bar. Replaced the 9-step onboarding with a 4-step progressive flow. Fixed 47 biometric authentication edge cases.

Tech Stack
React NativeTypeScript FigmaJest
Year2023
RoleLead UX Engineer
↗ App Store ⌥ GitHub

Key Outcomes

  • App Store rating improved from 2.9 to 4.8 stars within 8 weeks
  • Onboarding completion rate increased from 32% to 89%
  • Daily active users grew by 40% over the following quarter
