Project Overview

Open-source CLI tool that scaffolds opinionated frontend project structures, enforces code quality standards via automated linting, and generates component boilerplate. Downloaded over 180,000 times on npm.

Challenge

Every new project required the same 2-hour setup ritual: configuring ESLint, Prettier, Husky, lint-staged, TypeScript, and directory structure. With teams of 20+ developers, inconsistency led to merge conflicts and CI failures.

Solution

Extracted the setup into a single CLI command with a plugin architecture supporting React, Vue, and Vanilla TS. Interactive prompts via Inquirer.js for customisation. Published to npm and open-sourced with full documentation.
