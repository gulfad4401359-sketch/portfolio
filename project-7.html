Open Source CLI Tool

Node.jsOpen Source CLIDeveloper Tools

Jan 2021 — Ongoing

Project Overview

Built and maintained an open-source CLI tool that scaffolds opinionated frontend project structures, enforces code quality standards via automated linting and formatting, and generates component boilerplate. The tool has been downloaded over 180,000 times on npm.

Challenge

Every new project at my previous employer required the same 2-hour setup ritual: configuring ESLint, Prettier, Husky, lint-staged, TypeScript, and directory structure. With teams of 20+ developers, inconsistency in setup led to merge conflicts, CI failures, and onboarding friction for new hires.

Solution

Extracted the setup process into a single CLI command. Designed a plugin architecture to support React, Vue, and Vanilla TS targets. Added interactive prompts using Inquirer.js to customise scaffolding without requiring documentation. Published to npm and open-sourced on GitHub with full documentation site.
